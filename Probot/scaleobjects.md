Scaling objects is the primary function of Smart! v22!

An original feature of Smart! (loosely copied by Coffee Stain as "Zoop"), Smart! creates many copies of objects in all three dimensions!

--How to Scale--

1. Select an object in the build menu. You'll see a hologram.

2. Use NumPad keys to adjust grid size:
   - **NumPad 8/5** - X axis (width/rows)
   - **NumPad 6/4** - Y axis (depth/columns)
   - **NumPad 9/3** - Z axis (height/layers)

3. **Alternate:** Hold modifier keys + scroll/NumPad:
   - **X** + scroll/NumPad 8/5 - X axis
   - **Z** + scroll/NumPad 8/5 - Y axis
   - **X+Z** (both) + scroll/NumPad 8/5 - Z axis

4. **Recipe Selection:** Hold **U** + scroll/NumPad 8/5 to cycle recipes. HUD shows inputs/outputs. Type !recipe for details.

--Additional Features--

- **Spacing:** Hold **;** + scroll/NumPad 8/5 to add gaps. Press **Num0** (while holding **;**) to cycle X/Y/Z axes. Type !spacing for details.

- **Steps:** Hold **I** + scroll/NumPad 8/5 for vertical stairs. Press **Num0** (while holding **I**) to cycle X/Y axes. Type !steps for details.

- **Stagger:** Hold **Y** + scroll/NumPad 8/5 for diagonal/lean patterns. Press **Num0** (while holding **Y**) to cycle X/Y/ZX/ZY axes. Type !stagger for details.

--Keybinds--

All keybinds customizable via **Options > Keybindings > Smart! Scaling Controls**. Type !keybinds for more info.

--Scaling vs Extend--

Scaling creates grid copies (recipes auto-copied). Extend (!extend) copies single buildings with all splitters, mergers, belts, and connections. Extend planned for Phase 3.
