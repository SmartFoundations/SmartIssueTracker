[user]

All Smart! keybinds can be customized in-game via **Options > Keybindings > Smart! Scaling Controls** in Satisfactory's main menu.

--Default Keybinds--

**Grid Scaling:**
- **NumPad 8/5** - Increase/decrease X axis (width/rows)
- **NumPad 6/4** - Increase/decrease Y axis (depth/columns)
- **NumPad 9/3** - Increase/decrease Z axis (height/layers)
- **X** (hold) + scroll/NumPad 8/5 - X axis (alternate)
- **Z** (hold) + scroll/NumPad 8/5 - Y axis (alternate)
- **X+Z** (hold both) + scroll/NumPad 8/5 - Z axis (alternate)

**Spacing Mode:**
- **;** (hold) - Activate spacing mode
- Scroll/NumPad 8/5 (while holding **;**) - Adjust spacing
- **Num0** (while holding **;**) - Cycle X/Y/Z axes

**Steps Mode:**
- **I** (hold) - Activate steps mode
- Scroll/NumPad 8/5 (while holding **I**) - Adjust step height
- **Num0** (while holding **I**) - Cycle X/Y axes

**Stagger Mode:**
- **Y** (hold) - Activate stagger mode
- Scroll/NumPad 8/5 (while holding **Y**) - Adjust stagger
- **Num0** (while holding **Y**) - Cycle X/Y/ZX/ZY axes

**Recipe Selection:**
- **U** (hold) - Activate recipe mode
- Scroll/NumPad 8/5 (while holding **U**) - Cycle recipes
- **Num0** (while holding **U**) - Clear selection

**Visual Aids:**
- **NumPad 0** - Toggle axis direction arrows

--Customizing Keybinds--

1. Go to **Main Menu > Options > Keybindings**
2. Scroll to **Smart! Scaling Controls** section
3. Click on any keybind to remap it
4. Press your desired key to assign it

All keybinds are fully customizable to avoid conflicts with vanilla Satisfactory controls or other mods.
