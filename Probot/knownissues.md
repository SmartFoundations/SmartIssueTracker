--Limitations--

**Building Type Support:** Not all building types support all features. Resource extractors have scaling disabled. Some buildings have type-specific limitations.

**Grid Size Limits:** Maximum grid size varies by building type. Very large grids may hit game engine limits. Server validation may reject extremely large grids.

**Multiplayer Replication:** Some features may not replicate correctly. Host has full functionality; clients may experience limitations.

--Quick Workarounds--

**Scaling doesn't work?** Check if building supports scaling, verify keybinds are configured correctly.

**Spacing doesn't apply?** Verify spacing mode enabled (hold **;**), check counters, ensure correct axis.

**HUD doesn't display?** Verify mod loaded, check for UI conflicts, review logs.
