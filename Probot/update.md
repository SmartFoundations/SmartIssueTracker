-= Issues and solutions =-

1. Smart! v22 has just been released on SMM!

2. If you previously were using a -test version of Smart, be sure to select "Latest" again to ensure you are continuing to get the latest updates as they are released. Development is still active on Smart, so bugs will be addressed as they are discovered and the developer has time to look at it.

3. Make sure SMM is on the latest version!  In SMM, go to Settings -> SML Version -> Latest

4. Multi-player has issues for clients, long distances, etc: Multi-player is still not 100% supported. Type !multiplayer for more information.

5. Crashing or other issues with other mods: Did your game crash, and Smart is listed in the crash log? Report the issue on the GitHub with the details from the crash. Smart has not been tested with every combination of mods, and some can conflict. See !troubleshooting for more tips.

6. New default keybinds have been created for v22+! Type !keybinds in #botspam to get more details on how to work around it.

-= Status =-

v22 release is available as Phase 1 of Smart's return. A number of bugs are present, we are sure! Let us know! Please review any bugs you are experiencing, and if they not currently listed as known bugs, please use !bugreport and follow the instructions.
