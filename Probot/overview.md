Known Issues

Check patch notes for update-related issues that are known.

To find the complete list of known issues, please visit Link: <https://github.com/SmartFoundations/SmartIssueTracker/issues>

If you cannot find an issue that you expect to see, then please assume that it is not known and report it! For instructions, type !bugreport

When working on bug fixes, the priority queue is pulled exclusively from the GitHub issues. If they were mentioned in the Discord chat, but no one filed a bug report, it is likely it will not be reviewed!
